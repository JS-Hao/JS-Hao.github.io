插入

二叉查找树的插入规则如下：

  1. 判断BST对象是否有根节点，如果没有，则将“插入节点”（即将要插入的节点）插入到根节点处（root），否则将根节点设置为“当前节点”，进入步骤2
  2. 比较当前节点与插入节点的大小，若插入节点小于当前节点，则进入步骤3，大于则进入步骤4
  3. 访问当前节点的左节点，若为null，则插入到当前节点的左节点，若有，则将当前节点的左节点设置为当前节点，重复步骤2
  4. 访问当前节点的右节点，若为null，则插入到当前节点的右节点，若有，则将当前节点的右节点设置为当前节点，重复步骤2
